Everything you need to know about QuickBlog's powerful features

Can I publish blogs both via website and CLI?

+

Yes! Use the website for quick publishing with our web editor, or use our professional CLI tool for terminal-based workflow. Both publish to the same platform with instant live links.

What's the difference between website and CLI publishing?

+

Website: Create account, manage all your blogs forever, edit/delete anytime. CLI: Session-based, beautiful terminal UI with rainbow ASCII art, perfect for developers who prefer command-line workflows.

How does account management work on the website?

+

Sign up once, manage all your blogs forever. Edit, delete, or update any blog you've published. No session limits - full control over your content with persistent account access.

What's the backend architecture powering both?

+

Node.js + Express server with Redis Cloud database, deployed on Vercel. Full REST API serves both the website and CLI. Same data, same performance, different interfaces.

Why choose CLI over the website?

+

CLI offers beautiful terminal UI with figlet ASCII art, gradient text, professional tables, and developer-focused workflow. Perfect for those who live in the terminal and want gorgeous command-line blogging.

How reliable is the platform?

+

We guarantee 99.9% uptime with Redis Cloud enterprise infrastructure. Your content is replicated across multiple data centers. We've never lost a single post in our 3-year history.
